# NOVA Mobile App - Product Requirements Document (PRD)

## 1. Overview
NOVA is a modern mobile application designed to provide a seamless marketplace experience for buyers and sellers, focusing on trust, ease of use, and clean UI/UX.

## 2. Core Features
- **User Authentication**
  - Email/Password login
  - Phone number login (optional future)
  - Forgot password
- **Buyer Registration Flow**
  - Minimal onboarding
  - Profile creation after sign-up for additional details
- **Seller Registration Flow**
  - KYC-structured onboarding
  - Business verification
- **Product Listings**
  - Add, edit, delete
  - Image upload
  - Categorization
- **Search & Filters**
  - Smart search
  - Category filter
- **Messaging**

- **Payments (Future)**
